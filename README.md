# 🐘 Learning PHP
My personal journey through PHP development, from basics to advanced concepts.

## 🚀 What's Inside?
- **Basics:** Syntax, Variables (`$`), and Data Types.
- **Logic:** Conditionals (`if/else`), Loops, and Functions.
- **Arrays:** Indexed, Associative, and Multidimensional arrays.
- **Superglobals:** Working with `$_GET`, `$_POST`, and `$_SESSION`.
- **Database:** Connecting to MySQL using PDO/MySQLi.

## 🛠️ Environment
- **Server:** Apache (XAMPP / LocalWP)
- **Language:** PHP 8.x
- **Database:** MySQL
